## Documentation Standards

All documentation follows consistent patterns:

**Required Elements:**

- Metadata headers (Last Updated, Document Type, Audience)
- Brief intro paragraph
- Section dividers (`---`)
- Code blocks with language specification
- Practical examples

**Document Types:**

- **Guide**: Comprehensive learning resource with examples
- **Reference**: Quick lookup with consistent formatting
- **Quick Start**: Step-by-step getting started instructions
